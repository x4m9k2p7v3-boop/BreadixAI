### 5. ✅ Оптимизированы imports в HTML

**Файлы:** `index.html`, `sign_in.html`, `sign_up.html`

**Что сделано:**
- Добавлены секционные комментарии
- Группировка по слоям (External → Config → Core → Database → Features → Main)
- Понятен порядок загрузки и зависимости

**Было:**
```html
<script src="marked.min.js"></script>
<script src="config.js"></script>
<script src="prompts.js"></script>
...
```

**Стало:**
```html
<!-- === EXTERNAL LIBRARIES === -->
<script src="marked.min.js"></script>

<!-- === CONFIGURATION === -->
<script src="config.js"></script>
<script src="prompts.js"></script>

<!-- === CORE UTILITIES === -->
<script src="crypto-utils.js"></script>
...
```

**Польза:**
- Понятно, зачем нужен каждый скрипт
- Легко найти нужный импорт
- Видны зависимости между модулями
