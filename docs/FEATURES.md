# Новые функции BreadixAI

## 🔒 Безопасность

### 1. Защита от XSS
- Автоматическая санитизация всех пользовательских вводов
- Удаление опасных символов и скриптов

### 2. CAPTCHA
- Простая математическая капча после 5 неудачных попыток входа
- Защита от автоматизированных атак

### 3. Восстановление пароля
- Контрольный вопрос при регистрации
- Трёхэтапный процесс восстановления:
  1. Ввод логина
  2. Ответ на контрольный вопрос
  3. Установка нового пароля
- Страница: `password_recovery.html`

### 4. Смена пароля
- Доступна в настройках → Безопасность
- Требует ввода текущего пароля
- Валидация нового пароля

### 5. История активности
- Логирование всех действий пользователя
- Отображение последних 10 событий
- Информация: действие, время, платформа
- Доступна в настройках → Безопасность

### 6. Rate Limiting
- Максимум 5 попыток входа за 15 минут
- Автоматическая блокировка при превышении
- Защита от брутфорса

## 👤 Аккаунт

### 7. Аватарки пользователей
- Загрузка своего изображения (макс. 2MB)
- Отображение в меню пользователя

- Доступно в настройках → Аккаунт

- Полное удаление всех данных пользователя
- Подтверждение через модальное окно
- Доступно в настройках → Данные → Опасная зона

## 💾 Управление данными

### 9. Экспорт чатов
- Сохранение всей истории в JSON файл
- Формат: `breadixai_chats_username_YYYY-MM-DD.json`
- Горячая клавиша: `Ctrl + E`
- Доступно в настройках → Данные

### 10. Импорт чатов
- Загрузка ранее экспортированных чатов
- Автоматическое слияние с существующими
- Проверка формата данных
- Доступно в настройках → Данные

## 🔐 Улучшенная валидация

### Логин:
- Минимум 3 символа, максимум 20
- Только латинские буквы, цифры, дефис и подчёркивание
- Не может начинаться с цифры

### Пароль:
- Минимум 8 символов, максимум 64
- Обязательно: заглавная буква (A-Z)
- Обязательно: строчная буква (a-z)
- Обязательно: цифра (0-9)
- Обязательно: спецсимвол (!@#$%^&* и т.д.)
- Проверка на слабые пароли
- Индикатор надёжности в реальном времени

### Хеширование:
- Пароли хранятся в зашифрованном виде
- Используется хеш-функция с солью
- Невозможно восстановить исходный пароль

## 🚀 Как использовать

### Восстановление пароля:
1. На странице входа нажмите "Забыли пароль?"
2. Введите свой логин
3. Ответьте на контрольный вопрос
4. Установите новый пароль

### Голосовой ввод:
1. Нажмите на иконку микрофона рядом с полем ввода
2. Произнесите текст
3. Текст автоматически добавится в поле ввода

### Экспорт/Импорт:
1. Откройте настройки (Ctrl + ,)
2. Перейдите на вкладку "Данные"
3. Нажмите "Экспортировать чаты" или "Импортировать чаты"

### Статистика:
1. Откройте настройки
2. Перейдите на вкладку "Статистика"
3. Просмотрите детальную информацию об использовании
