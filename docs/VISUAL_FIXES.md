## Технические детали:

### Использованные техники:
- `flex-shrink: 0` — для иконок и кнопок
- `overflow: hidden` + `text-overflow: ellipsis` — для текста
- `white-space: nowrap` — против переноса
- `min-width: 0` — для правильного flex overflow
- `max-width` — для ограничения ширины
- `flex: 1` — для растягивающихся элементов

### Файлы:
- `visual-fixes.css` — все исправления в одном файле
- Подключен после основных стилей для переопределения
