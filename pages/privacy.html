Политика конфиденциальности

Последнее обновление: 19 апреля 2026

Важно: Мы серьёзно относимся к защите ваших данных. Эта политика объясняет, какую информацию мы собираем и как её используем.

1. Какие данные мы собираем

1.1 Данные аккаунта

  • Логин (имя пользователя)
  • Пароль (хранится в зашифрованном виде в localStorage браузера)

1.2 Данные использования

  • История ваших чатов с AI-моделями
  • Выбранные модели и настройки
  • Прикреплённые файлы и изображения

1.3 Технические данные

  • Логи запросов к API (для отладки и улучшения сервиса)
  • Информация о браузере и устройстве (автоматически)

2. Как мы используем данные

  • Для предоставления доступа к AI-моделям
  • Для сохранения истории ваших чатов
  • Для улучшения качества сервиса
  • Для технической поддержки и решения проблем

3. Хранение данных

Локальное хранение: Ваши данные аккаунта и история чатов хранятся локально в вашем браузере (localStorage). Мы не храним эти данные на наших серверах.

  • API ключи хранятся только на сервере в зашифрованном виде
  • История чатов хранится в localStorage вашего браузера
  • Прикреплённые файлы обрабатываются локально и не сохраняются на сервере

4. Передача данных третьим лицам

Мы передаём ваши запросы следующим сервисам:

  • OmniRoute API — для обработки запросов к AI-моделям
  • Tavily API — для веб-поиска (только при включении функции поиска)

Мы не продаём и не передаём ваши личные данные третьим лицам в коммерческих целях.

5. Безопасность

  • API ключи скрыты от клиента и хранятся только на сервере
  • Все запросы к API проходят через наш защищённый прокси-сервер
  • Мы используем HTTPS для защиты передачи данных
  • Пароли не хранятся в открытом виде

6. Ваши права

Вы имеете право:

  • Удалить свой аккаунт и все связанные данные
  • Очистить историю чатов в любой момент
  • Запросить информацию о хранящихся данных
  • Отказаться от использования сервиса

7. Cookies и localStorage

Мы используем localStorage для:

  • Хранения данных аккаунта
  • Сохранения истории чатов
  • Запоминания настроек (выбранная модель, тема)

Вы можете очистить localStorage через настройки браузера.

8. Изменения в политике

Мы можем обновлять эту политику конфиденциальности. О значительных изменениях мы уведомим пользователей через платформу.

9. Контакты

Если у вас есть вопросы о конфиденциальности, свяжитесь с нами: @Breadixx
